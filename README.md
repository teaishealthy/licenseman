# licenseman

A CLI tool to manage licenses in your projects.

## Installation

Download and install `terminaltables`:

```bash
pip install terminaltables
```

Now you can add `licenseman` to your `PATH` and use it.

## Usage

```bash
licenseman Canada
```

```bash
licenseman --spdx MIT
```
